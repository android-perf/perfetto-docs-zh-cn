# Perfetto doc 中文文档

[Perfetto docs](https://perfetto.dev/docs) 文档的中文翻译。

## 项目文档

.project 目录下为项目文档：
- **README.md** - 项目介绍
- **DEPLOYMENT.md** - 部署指南和故障排除
- **deploy.sh** - 自动化部署脚本
- **TRANSLATION_GUIDE.md** - 翻译规范

## 翻译

为确保高效和一致性，本项目使用 AI 工具进行翻译。翻译规范请参考 [翻译规范](TRANSLATION_GUIDE.md)。

工作流：
1. 更新上游 Perfetto 文档，检查同步节点
2. 使用 AI 工具按照翻译规范进行翻译
3. 人工校对，修正翻译结果或翻译规范
5. 提交更新
6. 部署更新
